# -Movie-Recommender-System-using-Machine-Learning
Movie Recommender System  Developed a Machine Learning based Movie Recommender System using Python, Pandas, Scikit-learn, and Streamlit. The system recommends similar movies based on content similarity using cosine similarity techniques. Integrated TMDB API to fetch real-time movie posters and built an interactive web interface using Streamlit.
